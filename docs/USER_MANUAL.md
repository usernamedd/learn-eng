# Learn English - 用户手册

跨平台英语学习助手，帮你快速搞定工作中遇到的不认识的英语单词和表达。

---

## 快速上手

### 截图学习（核心功能）

三步完成英语单词学习，操作成本极低：

```
截图学习 → AI 分析 → 查看结果卡片
```

#### 第一步：截图上下文

把你**正在看的页面内容**截下来，包含你不认识的单词所在的整体内容。

支持 **3 种输入方式**：

| 方式 | 操作 |
|------|------|
| 📷 **屏幕截图** | 点击「屏幕截图」按钮 → 全屏截图 → 拖动鼠标选择区域 → 点击「确认选择」 |
| 📋 **Ctrl+V 粘贴** | 用系统截图工具（微信截图 / Snipaste / 系统自带）截好图后，直接按 Ctrl+V 粘贴 |
| 📂 **拖拽图片** | 从文件夹或浏览器拖拽图片到指定区域 |

> 💡 **推荐方式**：用系统自带的截图快捷键截好图，回 App 里 Ctrl+V 粘贴，最快！

#### 第二步：截图目标词

把你不会的**那个单词或表达**所在的区域截图（特写，圈出目标）。

同样支持屏幕截图 / Ctrl+V 粘贴 / 拖拽三种方式。

#### 第三步：查看学习结果

AI 分析完成后，返回一张结构化**学习结果卡片**，包含 8 项内容：

| # | 内容 | 说明 |
|---|------|------|
| ① | **上下文释义** | 目标词在当前上下文中的具体含义，不是孤立的词典解释 |
| ② | **中文翻译** | 对应的中文说法 |
| ③ | **选词分析** | 为什么用这个词而不用其他近义词，强调什么语气 |
| ④ | **类似用法** | 同样结构在不同场景的举例 |
| ⑤ | **发音** | 音标 + 🔊 发音朗读 |
| ⑥ | **原句提取** | 从截图中提取包含目标词的完整句子，高亮目标词 |
| ⑦ | **词性 + 常用度标签** | 如 `📘 学术词汇` / `💼 商务词汇`、`⭐⭐⭐ 高频` / `⭐ 低频` |
| ⑧ | **一键收藏** | 保存到生词本，方便回顾 |

**结果卡片示例：**

```
┌─────────────────────────────────────────────┐
│  leverage                    🔊 /ˈlevərɪdʒ/ │
│  verb · ⭐⭐⭐ 高频 · 📘 学术词汇         ★ │
│                                             │
│  📖 利用（已有资源）                        │
│                                             │
│  💬 「The system will leverage existing     │
│   infrastructure to reduce costs…」         │
│              ⬆️                             │
│                                             │
│  💡 这里用 leverage 比 use 更正式，强调      │
│    "巧妙地利用已有资源来达到目的"             │
│                                             │
│  📝 类似用法                                │
│  • leverage data → 利用数据                 │
│  • leverage network → 利用网络              │
│  • leverage experience → 利用经验           │
└─────────────────────────────────────────────┘
```

查看完结果后可以：
- **🔊 点击小喇叭** → 发音朗读
- **☆ 收藏到生词本** → 保存结果，日后回顾
- **🔄 再来一次** → 重新截图查询

## 功能页面

### 📸 截图学习（首页）

默认进入页面，核心功能的完整三步流程如上所述。

### 🔤 翻译模式

说一句中文，翻译成目标语言。

1. 输入中文文字（或点击 🎤 语音输入）
2. 点击「翻译」按钮
3. 查看翻译结果，可点击 🔊 朗读译文

### 💬 对话模式

用中文说话，AI 用目标语言回复，文字 + 语音同步输出。

1. 点击 🎤 按钮开始语音输入
2. AI 用目标语言回复
3. 可点击 🔊 朗读每条 AI 回复

### 📚 生词本

所有在截图学习中收藏的单词自动保存到这里。

- 列表展示：**目标词** + **中文释义** + **收藏时间**
- 点击条目 → 展开查看完整学习结果卡片
- 支持删除条目

### ⚙️ 设置

| 配置项 | 说明 |
|--------|------|
| 🎯 目标语言 | 学习的目标外语（默认 English） |
| 🤖 AI 模型 | LLM 提供商 + API Key |
| Base URL | 自定义 API 地址（Ollama / vLLM 等需要） |

## 使用场景

### 场景一：看英文博客

```
你正在阅读一篇英文技术博客 → 遇到"leverage"这个词
→ 截上下文（博客页面）→ 截目标词（leverage 所在行）
→ AI 告诉你：这里用 leverage 强调"利用现有资源"
→ 🔊 听发音 → ⭐ 收藏 → 继续阅读
```

### 场景二：看英文文档 / API

```
遇到一段英文文档看不懂 → 截图上下文 → 截图目标句
→ AI 给出中文翻译 + 词法分析 → 秒懂
```

### 场景三：IDE 里写代码

```
IDE 里的注释/报错有英语单词不认识
→ 截图 → 粘贴 → 学习 → 继续 coding
不离开当前工作流
```

## 常见问题

**Q: 截图功能需要什么权限？**
A: 首次点击「屏幕截图」按钮时，系统会弹出屏幕录制/截图权限请求，需要授予权限。

**Q: 如果截图太大怎么办？**
A: 裁剪时可以只选择包含目标词的小区域，AI 有足够的上下文理解。

**Q: 支持哪些语言？**
A: 默认英语（en），可在设置中切换日/韩/德/法/西等语言。截图分析功能支持所有语言。

**Q: 需要什么 AI 模型？**
A: 截图学习需要支持视觉（Vision）能力的模型，推荐 DeepSeek-Chat（内置默认）、GPT-4o、Claude-3。对话和翻译模式普通模型即可。

**Q: 生词本存在哪里？**
A: 生词本保存在本地浏览器存储中（localStorage），不同电脑之间不共享。
