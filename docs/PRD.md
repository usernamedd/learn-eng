# 英语学习助手 - 需求文档

## 一、背景与目标

帮助用户通过翻译练习和对话练习提升英语水平。程序提供两种工作模式，用户可以在应用中实时切换。

---

## 二、用户需求层次

### 2.1 翻译练习需求
- 用户想说一句中文，程序能翻译成目标语言（默认英文，可切换其他语言）
- 用户可以说一句完整的句子，得到一句翻译结果
- 翻译结果以文字形式展示给用户

### 2.2 对话练习需求
- 用户用母语（中文）说话，程序用目标语言与用户对话交流
- AI 回复不仅显示文字，还通过语音朗读出来
- 用户可以选择 AI 使用的目标语言（默认英文，可切换）

### 2.3 灵活性需求
- 目标语言可自由切换，切换后两种模式立即生效
- AI 模型可自由切换（云服务或本地部署）
- API Key、模型参数等可在应用内配置
- 应用提供配置管理界面

---

## 三、功能需求层次

### 3.1 核心功能

#### F1：翻译模式
- 用户通过语音输入中文句子
- 程序识别语音内容，转为文字
- 调用大模型翻译成目标语言
- 将翻译结果以文字形式展示在界面上
- 支持目标语言切换（英文、日文、韩文、德文、法文等）

#### F2：对话模式
- 用户通过语音输入内容（中文）
- 程序识别语音内容，转为文字
- 调用大模型，以目标语言与用户对话
- 将 AI 回复以文字形式展示
- 将 AI 回复以语音形式朗读出来（文字转语音）
- 支持目标语言切换

#### F3：语言切换
- 在应用主界面提供目标语言切换入口
- 切换目标语言后，翻译模式和对话模式均使用新语言
- 显示当前选中的目标语言

### 3.2 配置功能

#### F4：模型配置
- 支持选择 AI 服务提供商（DeepSeek / OpenAI GPT / Claude / Gemini / Ollama / vLLM）
- 支持配置各提供商的 API Key
- 支持配置自部署模型的地址（如 Ollama、vLLM 的地址）

#### F5：配置管理界面
- 在应用内提供配置页面
- 可查看和修改当前配置
- 配置持久化保存

### 3.3 技术需求

#### F6：语音识别
- 调用系统或第三方语音识别能力，将用户语音转为文字
- 支持中文语音输入

#### F7：语音合成（TTS）
- 将 AI 回复的文字转换为语音
- 使用目标语言的发音

---

## 四、非功能需求

- 各平台独立开发，共享核心业务逻辑
- 核心业务代码可在无 UI 环境下单独测试
- 应用需有友好的交互反馈（加载状态、错误提示）

---

## 五、验收标准

- [ ] 用户可以选择目标语言，切换后两种模式均使用新语言
- [ ] 翻译模式：用户说一句中文，程序显示对应目标语言翻译
- [ ] 对话模式：用户说一句中文，程序用目标语言回复，显示文字并朗读语音
- [ ] 用户可以在应用内配置 AI 模型和 API Key
- [ ] 配置修改后，下次请求使用新配置
- [ ] 翻译模式和对话模式可随时切换
