# 英语学习助手 - 需求文档

## 一、背景与目标

用户在使用电脑（浏览器、IDE、文档、PDF 等）的过程中遇到不认识的英语单词或表达，
通过截图的方式快速学习。**操作成本极低，不中断当前工作流。**

---

## 二、用户需求

### 2.1 截图学习

用户在电脑前工作时，遇到不认识的英语单词或表达，操作方式如下：

1. 用户截**两张图**：
   - **第一张截图（上下文）**：当前屏幕的上下文内容，可能包含大量文字、图片、代码等。
     **一张截图就能包含所有上下文信息**，不需要多张截图拼凑。
   - **第二张截图（目标）**：用户不熟悉的那个单词、表达或词汇所在的位置。
2. 程序对两张截图进行分析：
   - 从第二张截图中识别出目标词/表达
   - 结合第一张截图的上下文理解该词在具体场景中的含义

## 三、学习结果卡片

程序返回一个结构化的结果卡片，包含以下 8 项内容：

| # | 内容 | 说明 |
|---|------|------|
| ① | **上下文释义** | 目标词在这个具体上下文中的含义，而非孤立的词典解释 |
| ② | **中文对应说法** | 对应中文翻译 |
| ③ | **选词分析** | 为什么用这个词而不用其他近义词，强调什么语气/风格 |
| ④ | **类似用法** | 同样结构在不同场景的举例（如 leverage data、leverage network） |
| ⑤ | **发音** | 音标 + 发音朗读 |
| ⑥ | **原句提取** | 从上下文截图中提取包含目标词的完整句子，高亮目标词 |
| ⑦ | **词性 + 常用度标签** | 如 `📘 学术词汇` / `💼 商务词汇` / `📱 日常`，以及 `⭐⭐⭐ 高频` / `⭐ 低频` |
| ⑧ | **一键收藏** | 将本次查询（截图 + 结果 + 原句）保存到生词本 |

### 结果卡片示例

```
┌──────────────────────────────────────────┐
│  leverage                      🔊  /ˈlevərɪdʒ/  │
│  📘 学术词汇 · ⭐⭐⭐ 高频         ★ 收藏     │
│                                            │
│  📖 利用（已有资源）                       │
│                                            │
│  「The system will leverage existing       │
│   infrastructure to reduce costs…」        │
│              ⬆️                            │
│                                            │
│  💡 这里用 leverage 比 use 更正式，强调     │
│  "巧妙地利用已有资源或优势来实现目标"        │
│                                            │
│  📝 类似用法                               │
│  • leverage data → 利用数据                │
│  • leverage network → 利用网络             │
│  • leverage experience → 利用经验          │
└──────────────────────────────────────────┘
```

## 四、生词本

- 所有收藏的查询自动保存到生词本
- 生词本列表展示：目标词、释义、收藏时间、来源截图（缩略图）
- 可点击展开查看完整结果卡片
- 支持按时间排序、按常用度筛选

---

## 五、非功能需求

| 需求 | 说明 |
|------|------|
| **操作路径短** | 截图 → 出结果，两步完成，无需额外打字 |
| **响应快** | 从截图到出结果不超过 5 秒 |
| **不中断工作流** | 截完图拿到结果，继续干自己的事 |
| **多种输入方式** | 支持屏幕截图、图片文件拖入、剪贴板粘贴 |
| **截图格式** | 支持 PNG / JPG / WebP 等常见格式 |

---

## 六、验收标准

- [ ] 用户截两张图（上下文 + 目标），程序正确识别目标词
- [ ] 结合上下文给出准确释义，而非孤立的词典定义
- [ ] 结果卡片包含全部 8 项内容
- [ ] 从截图到出结果不超过 5 秒
- [ ] 发音可播放
- [ ] 一键收藏，生词本完整保留查询记录
- [ ] 生词本可查看和回顾
- [ ] 支持多种输入方式（截图/拖入/粘贴）
